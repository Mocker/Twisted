*Python:     http://www.python.org/
  
*PyGame:     http://www.pygame.org/
  





RUNNING THE GAME:

open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:


Use arrows keys to move around. Space to jump / select menu items.
